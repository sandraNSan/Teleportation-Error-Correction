# TEC: Enhanced Error Correction for Noisy Teleported States under the Depolarising Noise Model

Simulation code and supporting materials for analysing **teleportation based
error correction (TEC)** under noisy quantum channels.

---

## Overview

This repository contains simulation code and supporting materials for studying
**teleportation based error correction (TEC)** in noisy quantum communication
systems. The focus is on mitigating the effects of noise during
quantum teleportation using stabiliser measurements and error‑correcting codes.

---

## Simulation Software Environment

The simulations were developed and tested using the following environment:

- **Python**: 3.13.9  
- **Qiskit**: 2.3.0  

> ⚠️ Using different versions may lead to numerical differences or software errors.
